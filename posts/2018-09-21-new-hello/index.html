前言

这里是二心的第不知道多少个博客。

初中那会，在各大门户网站上都建了博客，随手写一点东西大概是那会养成的习惯。

暂时决定微信公众号可能会更新编程之外的东西，这里会主要更新关于编程的一些思考和记录。

最近的学习重心大概有以下几点：

  • Vue 基本语法回顾
  • Vue 练手项目的完善
  • React Demo 回顾
  • 大全的内容过一遍

所谓写字即是露怯，是指写出来的东西总会被指摘，被当做讨论的根据，即使是自己，多年后看到当时写的内容也多半难以接受， 但好处还是更多一些的，所以我选择写一点。
